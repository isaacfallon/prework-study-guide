Git

Git flow steps:

  1. git status: checks what branch we are currently on
  2. git checkout main: switches to the main branch
  3. git pull origin main: pulls down the latest main branch
  4. git checkout -b 'branch-name': creates a new branch and switches to it [use git status to ensure you're working on a branch]
  5. Make your changes to the code
  6. git add -A: commit all files in the working branch
  7. git commit -m "add change notes here": commit the change with a message detailing the changes
  8. git pull origin main: pulls in the base branch [good habit to pull the base branch into your feature branch before pushing up any code]
  9. git push origin 'branch-name': pushes all changes to github
