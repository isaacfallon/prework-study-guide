HTML

  • The head element contains information about the webpage.
  • Set the charset with a meta element within the head element. This will usually be: 'UTF-8' as it is the dominant encoding for the World Wide Web.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

Git flow steps:

  • git status: checks what branch we are currently on
  • git checkout main: switches to the main branch
  • git pull origin main: pulls down the latest main branch
  • git checkout -b 'branch-name': creates a new branch and switches to it [use git status to ensure you're working on a branch]
  • Make your changes to the code
  • git add -A: commit all files in the working branch
  • git commit -m "add change notes here": commit the change with a message detailing the changes
  • git pull origin main: pulls in the base branch [good habit to pull the base branch into your feature branch before pushing up any code]
  • git push origin 'branch-name': pushes all changes to github

Javascript

    • A variable is a named container that allows us to store data in our code.
    • Control flow is the order in which a computer executes code in a script.
    • A JavaScript function is a block of code designed to perform a particular task.
